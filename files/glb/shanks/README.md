# Shanks (Ring Bands) Documentation

This directory contains the 3D models for the ring bands (settings) used in the Fana Jewelry configurator.

## Organization

The shanks are organized into three subdirectories based on their **Base Size**. Each base size is optimized to fit a specific range of center stone carats:

1. **[0.75 Carat Base](./0.75%20Carat%20Base)**: Best for stones between 0.50ct and 1.25ct.
2. **[2.00 Carat Base](./2.00%20Carat%20Base)**: Best for stones between 1.50ct and 2.50ct.
3. **[3.00 Carat Base](./3.00%20Carat%20Base)**: Best for stones 3.00ct and above.

## Naming Convention

Files follow the pattern: `{StyleID}-{BaseSize}.glb`
Example: `SP1001-075.glb`
- `SP1001`: Style ID (The unique design of the band)
- `075`: Base Size (Indicates it was modeled for a 0.75ct stone)
