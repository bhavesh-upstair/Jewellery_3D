# Heads (Center Stones) Documentation

This directory contains the 3D models for the center stones (diamonds) used in the Fana Jewelry configurator.

## Organization

The heads are organized by **Diamond Shape**:

- **[Round](./Round)**
- **[Emerald](./Emerald)**
- **[Oval](./Oval)**
- **[Pear](./Pear)**
- **[Princess](./Princess)**
- **[Marquise](./Marquise)**

## Naming Convention

Inside each shape folder, files are named by their **Carat Weight**:
Example: `1.00 Carat.glb`

## Usage in App

When loading a head model, you should select the shape folder and then the file corresponding to the desired carat weight.

Example path: `heads/Round/1.50 Carat.glb`
